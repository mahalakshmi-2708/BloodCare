**Description:** AI-powered Blood Donation Prototype that connects donors and patients, tracks donations, and provides educational support.

## How to Run
- Open `index.html` in a browser (frontend-only prototype)
- Navigate through dashboards: Home, Donate, Request Blood, Learn, History, Profile
